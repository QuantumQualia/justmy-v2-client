CityOS events (script embed)

Served from /embed/cityos.js (Vite IIFE + shadow DOM). The widget calls /api/embed/cityos-events on the script’s origin. By default the market domain is taken from window.location (the page hosting the script). Set data-domain when that host is not the market site. Optional data-events-limit (1–100).
